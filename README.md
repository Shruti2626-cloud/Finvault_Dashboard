# 💸 FinVault – Finance Dashboard UI

🔗 **Live Demo:** https://finvaultuserdashboard.vercel.app/

---

## 📌 Overview

FinVault is a modern, interactive **FinTech Dashboard UI** built using React and Tailwind CSS.
It enables users to track, analyze, and manage their complete financial ecosystem through a clean, visually rich, and highly interactive interface.

The project focuses on **frontend architecture, UI/UX design, and state management**, simulating real-world financial platforms using mock data, dynamic components, and smooth interactions.

---

## ✨ Features

### 📊 Dashboard Overview

* Total Balance, Income, and Expenses cards
* Monthly growth/decline indicators
* Balance trend & expense trend (interactive line charts)
* Spending breakdown with smooth animated pie chart

---

### 💳 Transactions Management

* Detailed transaction list (date, amount, category, type)
* Credit (green) & Debit (red) indicators
* Search, filtering, and sorting functionality
* Export transactions (CSV / JSON / PDF)

---

### 🧠 AI Insights & Analytics

* AI-based mock insights for spending behavior
* Future expense prediction based on trends
* Suggestions to optimize and reduce unnecessary expenses
* Monthly comparison and highest spending category detection

---

### 🏦 Investments Section

* Mock stock portfolio tracking
* Profit/loss visualization
* Simulated NIFTY/SENSEX indicators

---

### 💰 Loans Management

* Dedicated loans dashboard
* Track ongoing and completed loans
* EMI tracking (paid vs pending)
* Clear overview of liabilities and repayment progress

---

### 💳 Cards & Wallet System

* Virtual wallet interface
* Multiple card management
* Card limits and available balance tracking
* Payable amounts overview

---

### 📈 Reports & Financial History

* Monthly transaction reports
* Historical financial summaries
* Structured view of spending trends over time

---

### 👤 Profile & Settings

* User profile management
* Account settings interface
* Role-based UI (Admin / Viewer simulation)

---

### ⚡ Quick Actions

* Send Money, Deposit, Withdraw, Exchange
* Interactive popup-based workflows
* Smooth animations and transitions

---

### 🔔 Smart Notifications

* Dynamic notifications based on transaction data
* Real-time-like alerts for credits and debits

---

### 🎨 UI/UX Enhancements

* Glassmorphism design system
* Smooth animations using Framer Motion
* Hover effects & micro-interactions
* Fully responsive design (mobile, tablet, desktop)

---

### 🌙 Theme System

* Efficient Dark / Light mode toggle
* Custom bluish fintech dark theme
* Clean, premium light mode with proper readability

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **Charts:** Recharts
* **Animations:** Framer Motion
* **State Management:** Context API
* **Deployment:** Vercel

## ⚙️ Installation & Setup

```bash
git clone https://github.com/Shruti2626-cloud/Finvault_Dashboard.git
cd Finvault_Dashboard
npm install
npm run dev
```
